三、

1. 相关分析与回归分析有密切的联系，它们都是对变量之间相关关系的研究，二者可以相互补充。相关分析可以表明变量之间相关关系的性质和程度，只有当变量之间存在一定程度的相关关系时，进行回归分析去寻找相关的具体数学形式才有实际的意义。同时，在进行相关分析时，如果要具体确定变量之间相关的具体数学形式，又要依赖于在回归分析，而且相关分析中相关系数的确定也是建立在回归分析基础上的。
二者的区别：
	（1）在相关分析中，变量x和变量y处于平等的地位；在回归分析中，变量y称为因变量，处在被解释的地位，x称为自变量，用于预测因变量的变化。
	（2）相关分析中所涉及的变量x和y都是随机变量；在回归分析中，因变量 y是随机变量，自变量x可以是随机变量，也可以是非随机的确定变量。
	（3）相关分析主要是描述两个变量之间线性关系的密切程度；回归分析不仅可以提示变量x对y的影响大小，还可以由回归方程进行预测和控制。

2. 
（1）相关系数的取值范围为：-1<=r<=1
（2）r > 0，是正相关；r < 0，是负相关
（3）|r| 越接近0，相关程度越为不相关
（4）|r|＝1，为完全相关，
（5）|r|<0.3，为不相关或微弱相关，|r|越接近1，相关程度越高
	0.3 <= |r| < 0.5，为低度相关；
	0.5 <= |r| < 0.8，为显著相关；
	0.8 <= |r| < 1，为高度相关

3. 
（1）在两个变量之间必须根据研究的目的确定哪个是自变量，哪个是因变量
（2）在没有明显因果关系的两个变量中，可配合两个回归方程。值得注意的是，若两个变量存在明显的因果关系时，只能计算一条回归直线，另一条配合出来也没有意义
（3）回归方程的作用在于给出自变量的数值来估计因变量的可能值
（4）在直线回归方程中，自变量的系数b被称为回归系数。回归系数的符号为正时表示正相关，为负表示负相关
（5）在回归分析中，因变量是随机的，而把自变量当作研究时可以控制的量。即在给定不同自变量的数值下，观察对应的因变量数值的变化情况

4. 
（1）圆的面积与半径之间的关系是函数关系，面积是关径的系数
（2）工人的技术水平与产品质量之间的关系是相关关系
联系：函数关系和相关关系都表现为相互依存关系，一种现象的变化会引起另一种现象的变化。圆的面积随半径的变化而变化，产品质量随着工人的技术水平的提高而提高
区别：函数关系是非常严格的数量依存关系，某个现象的某个数值有另一个现象的完全确定的值与之对应，每给定一个半径就有一个唯一确定的圆面积与它对应。相关关系是非严格的存依存关系，某个现象的某个数值有另一现象的若干个值与之对应，技术水平完全相同的两名工人，他们加工出来的产品质量不一定完全相同。

5.
正相关：两个变量之间的变动方向是一致的。例如，随着施肥量的增加，平均亩产量也一般也会相应增加，施肥量与平均亩产量之间是正相关关系
负相关：两个变量之间的变动方向是相反的。例如：随着产品产量的增加，产品的单位成本是下降的，产品产量与单位成本之间是负相关关系
