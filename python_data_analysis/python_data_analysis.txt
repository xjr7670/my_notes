序列运算：
	二维数组或者序列array中，用0表示横向取值或运算——即横轴，1表示纵向取值或运算——即纵轴。
	如

	f = np.array([[1, 2, 3], [4, 5, 6]])
	>>> f
	array([[1, 2, 3],
		   [4, 5, 6]])

	如果执行f.sum(0)
	则会把第一行与第二行的数相加，得到结果
	array([5, 7, 9])

	如果执行f.sum(1)
	则会把第一列、第二列、第三列的数相加，得到结果
	array([6, 15])

	如果执行f.sum(2)
	则会提示轴（索引）超出范围

	实例方法或者np的顶级方法.cumsum()是对所有元素的累计求和
